# Ballerina Native APIs Distribution

This distribution contains the native APIs of Ballerina language.

Folder structure of the distribution;
```sh
ballerina-native-apis-<version>
├── LICENSE
├── README.md
└── ballerina
```
**ballerina:** is a folder which contains the native APIs of different Ballerina packages.
eg:
```sh
├── lang
│   ├── array
│   │   └── natives.bal
```
**lang/array/natives.bal:** contains the native APIs of the **ballerina.lang.array** package.
